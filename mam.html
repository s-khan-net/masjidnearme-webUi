Help

   When you start this application, it locates you on the map , and gets the nearest masjids , masjid names are listed on the menu. On touching the marker, or on a name from the menu, the details of the masjid are shown. You can navigate to the masjid by touching the button in the description window. (this will open google maps).
 Refresh the search by touching
on the bottom left
. You can move the pin (long press the pin to activate) to reload the masjids at the new location. Watch a demo here:
If no masjids are found, search for masjids will be triggered automatically by incrementing the search radius. Watch:

   The masjids loaded on the menu, can be verified (the ones present in our database) or unverified (the ones on google maps).
The verifed masjids have data about the Salaah times; more details that can be viewed by touching the masjid marker's infowindow
 Registered users with edit permissions can edit the details by using button, this will open the edit form. Watch: To verify a masjid, would have to use the Verify button that will navigate to masjidnear.me. Watch:

 Salah time notifications, location for notifications, Hanafi or Shafi setting, Search radius, Location setting for slaah times reminders, Salaah times calculation method & Number of masjids loaded, can be set using the Settings. Registered users can login using the Users menu. Unregistered user would have to use the website to register, The link in the sign up form, (Please use the website to Register) will directly take you there. watch this demo:
